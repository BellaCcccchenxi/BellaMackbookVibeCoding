# 每日诗词浮窗 · poetry-widget

Mac 桌面左上角常驻诗词浮窗：每日读取农历（干支八字 / 节气 / 节日）匹配推送一首经典古诗，清新古韵国风。

## 运行

```bash
cd poetry-widget
npm start        # 复用上级目录已安装的 electron
```

窗口出现在屏幕**左上角**，宽 = 屏幕 1/6，高 = 宽的 1/2，右上角有关闭「✕」。整窗可拖动。

## 匹配逻辑（almanac.js）

优先级：**传统节日 → 清明 → 当日节气 → 季节兜底**。
- 用 `lunar-javascript` 读取农历、干支、八字、生肖、节气、节日。
- 同一天用「年份 + 日干支」做稳定种子选诗，当天固定、跨日轮换（含八字意味）。
- 诗词库见 `poems.js`，已含春节/端午/中秋/重阳等节日、二十四节气、四季兜底名篇，可自行扩充。

## 国风设计

- 纯生成背景：宣纸米黄 + 水墨渐染 + 中式回纹细纹 + 角落梅枝，无任何真人肖像/版权素材。
- 半透宣纸面板 + 文字描边阴影，保证文字清晰适配背景。
- 书法字体用 macOS 自带：正文 `STKaiti`（楷体），标题/印章 `STXingkai`（行楷）。

## 可选自定义（不违反版权前提下）

- **自备背景图**：把图片（如你合法拥有的古风表情包）放入 `poetry-widget/assets/backgrounds/`，程序会按天轮换叠加为背景；文件夹为空时用生成式背景。
- **自备字体**：如需特定书法家字体，请自行确认授权后安装到系统，再在 `styles.css` 的 `--kai / --xing` 变量里加上字体名。

## 文件

| 文件 | 作用 |
|---|---|
| `main.js` | 窗口（无边框/透明/置顶/尺寸定位）、IPC |
| `preload.js` | 安全桥接 |
| `almanac.js` | 农历读取 + 诗词匹配 |
| `poems.js` | 诗词库（可扩充） |
| `index.html` / `styles.css` / `renderer.js` | 国风 UI |
